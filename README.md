# Sistema de Gestión de Clientes, Servicios y Reservas

## 📌 Descripción
Este proyecto implementa un sistema orientado a objetos en Python para la gestión de clientes, servicios y reservas.

El sistema está diseñado sin base de datos, utilizando estructuras internas y manejo de archivos para registrar eventos y errores.

---

## 🧠 Características

- Programación Orientada a Objetos (POO)
- Abstracción, herencia y polimorfismo
- Manejo avanzado de excepciones
- Excepciones personalizadas
- Registro de errores en archivo logs.txt
- Menú interactivo

---

## ⚙️ Funcionalidades

- Registrar clientes
- Crear servicios (Sala, Equipo, Asesoría)
- Crear reservas
- Visualizar reservas
- Manejo de errores sin detener el sistema
